Introduction

This is my first web site which is being hosted for FREE by Vercel.

It is linked to my GitHub repository; meaning, whenever I update the GitHub repository; then, those changes are also reflected here being seen, 'live'.

I was very impressed by the speed of setting up/deployment...; as it all took no more than just merely a few minutes. (Perhaps, less than 5 minutes.)

How I set up my GitHub repo to work with my Vercel a/c.

  1. Went to Vercel web site...where I used my email to sign up for an a/c.
  2. Next, I gave them my mobile phone number...so, they could sent me an ID code to type in.
  3. Recieved code on mobile phone...; came back to web browser and typed the code in.
  4. Used the Vercel UI/User Interface...to connect to my GitHub a/c.
  5. Next, connected my GitHub repo...to Vercel.
  6. Was given a URL to go and view the deployed site with.
